## 1일3제: 대소문자 구분 줍줍

https://www.acmicpc.net/problem/2744

```js
/**
 * @param {string} char
 * @returns {boolean}
 */
function isLowerCase(char) {
  if ('a'.charCodeAt() <= char.charCodeAt()) return true;
  else return false;
}
```
