## `:messages`로 로그보기

```
Word 'n' added to ~/.config/nvim/spell/en.utf-8.add
```

위와 같은 로그가 발생했는데 찝찝했었습니다. 그래서 nvim으로 로그를 어떻게 볼 수 있는지 검색했습니다.

```
:messages
```

위 커맨드로 로그를 확인할 수 있습니다.
