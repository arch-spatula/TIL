# Shadow DOM

```html
<input type="range" />
```

위와 같은 방식으로 작성하면 실제로 안에 div가 몇개 더 있습니다. 크롬 자체에서 보이지 않는 div가 있습니다. 설정으로 접근해서 볼 수 있습니다.
