# img tag의 display는 inline

```HTML
<h1></h1>
<p></p>
<div></div>
<img/>
```
위 태그 4개 중에 `display`가 `block`이 아는 것은 `<img/>`만 해당합니다. 오늘 레벨테스트하고 틀리면서 배웠습니다.
