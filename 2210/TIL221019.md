# todo tree 기본 설정

`[ ]`이렇게 되어 있으면 트레킹합니다. `[x]`이렇게 되면 완수로 봅니다. 하지만 계속 트레킹합니다. Extension setting에서 `[x]`은 삭제하도록 합니다. 마크다운은 완수로 표시하지만 트레킹을 덜 번거롭게 합니다.

- `[ ]` - 예정
- `[x]` - 완수
- `[-]` - 이월

이런 구분이 좋은 것 같습니다.

[]을 트레킹 안하고 `TODO`로만 트레킹하는 것도 좋겠습니다. `[-]`이 마크업 적용이 불가능하면 안 좋을 것 같습니다.
